# Quickstart

This repository is documentation-first. No external services or machine-specific infrastructure are required.

## Browse the Glossary

Open [GLOSSARY.md](GLOSSARY.md) and search for robotics, ROS 2, simulation, and AI terms.

## Review the SysML v2 Examples

The example AMR models live in [sysml_v2_models/](sysml_v2_models/):

- `bdd_amr_architecture.sysml`
- `ibd_amr_interconnects.sysml`
- `pkg_amr_decomposition.sysml`
- `uaf_reference_outline.md`

Use a SysML v2-capable editor or plain text review workflow.

## Extend the Glossary

1. Add a concise definition to [GLOSSARY.md](GLOSSARY.md).
2. Include the expanded acronym when applicable.
3. Add practical context without assuming an internal project or deployment.
4. Keep model references aligned with the SysML files.
