# Specification

## Goal

Publish a public robotics glossary and lightweight SysML v2 example set for robotics and MBSE discussion.

## Non-Goals

- Publish private project architecture, client/employer material, proprietary models, credentials, or unpublished roadmap notes.
- Claim the model set is authoritative, complete, approved for deployment, or suitable as a final system design.
- Include regulated, controlled, or private operational data.

## Public Boundary

This repository is a public reference layer. It may include glossary terms, usage notes, and small model examples suitable for public inspection.

## Success Criteria

- Terms and examples are clear, generic, and public-safe.
- Model examples remain lightweight and educational.
- Release checks pass before public updates.
