# Robotics Ontology

A public glossary and lightweight SysML v2 model set for robotics and Model-Based Systems Engineering (MBSE) examples.

This repository is intended as a reference for common robotics terms, ROS 2 concepts, simulation vocabulary, and a small Autonomous Mobile Robot (AMR) architecture example. It supports the public robotics portfolio by making terminology and architecture assumptions explicit.

## What This Is For

- Establish a shared vocabulary across robotics, ROS 2, and simulation work
- Provide lightweight SysML v2 examples for public MBSE discussion
- Give the rest of the portfolio a consistent terminology and architecture reference

## What It Demonstrates

- Shared vocabulary for robotics, simulation, ROS 2, and MBSE discussions.
- Lightweight SysML v2 textual modeling for an AMR-style system.
- A public reference layer that can be linked from robotics demos and case studies.

## Contents

- [GLOSSARY.md](GLOSSARY.md): robotics, ROS 2, simulation, and AI terminology.
- [USAGE.md](USAGE.md): guidance for extending the glossary and model set.
- [sysml_v2_models/](sysml_v2_models/): SysML v2 textual examples for AMR structure and interconnects.

## Use Cases

- Build a shared robotics vocabulary for teams and projects.
- Use the AMR model files as small MBSE examples.
- Cross-reference terminology from robotics demos, tutorials, or documentation.
