# SysML v2 Models

This directory contains small SysML v2 textual examples for an Autonomous Mobile Robot (AMR). The models demonstrate system decomposition, internal interconnects, and package-level structure.

## Included Models

- `bdd_amr_architecture.sysml`: block definitions for an AMR platform, compute node, sensor suite, motor controller, and power distribution unit.
- `ibd_amr_interconnects.sysml`: internal structure and connections between AMR components.
- `pkg_amr_decomposition.sysml`: package-level decomposition using SysML v2 textual notation.
- `uaf_reference_outline.md`: a lightweight mapping outline for architecture-framework thinking.

## Review Notes

These files are examples, not certified system designs. Review syntax and semantics with a SysML v2-capable tool before using them in production engineering work.
